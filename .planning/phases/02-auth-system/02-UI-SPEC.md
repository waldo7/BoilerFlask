## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gap in inline links; field hint text padding |
| sm | 8px | Compact element spacing; cross-link gap above divider; `.me-2` icon margins |
| md | 16px | Default element spacing; card body padding; form group stack gap; alert vertical rhythm |
| lg | 24px | Section padding; card-outer separation from brand header and back-to-home link; `.p-4` equivalent |
| xl | 32px | Layout gaps; auth card max-width margin from viewport edge |
| 2xl | 48px | Major section breaks; brand-header-to-card separation; flash-message-to-card gap |
| 3xl | 64px | Page-level spacing; vertical centering buffer on tall viewports |

Exceptions: none

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 500 | 1.4 |
| Heading | 24px | 600 | 1.3 |
| Display | 40px | 600 | 1.2 |

**Usage rules:**
- **Body:** All paragraph text, helper text, placeholder text, cross-link text, flash message body. Font size matches Bootstrap `$font-size-base`.
- **Label:** All form field labels (`<label>`, `form-label`). Smaller than body to distinguish from content.
- **Heading:** Card titles only ("Log In", "Create Account", "Forgot Password", "Set New Password", "Check Your Email", "Dashboard"). Matches Bootstrap `.h4` scale.
- **Display:** Auth shell brand header ("FlaskStuct") only. Matches `error.html` icon size visual weight. Applied via custom class `.auth-brand`.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f8f9fa` | Page background on auth shell (matching `error.html` precedent); desktop sidebar background; mobile topbar background |
| Secondary (30%) | `#ffffff` | Auth card surface (`.card`); form input backgrounds; alert banner backgrounds |
| Accent (10%) | `#0d6efd` | Primary action buttons ("Log In", "Create Account", "Send Reset Link", "Reset Password"); auth shell brand icon; form focus ring; sidebar `.active` nav link background; inline links in cross-link text |
| Destructive | `#dc3545` | Danger alert banners (invalid credentials, server-side errors); "Delete" or destructive confirmation buttons only (none in Phase 2; reserved for future) |

**Accent reserved for:** Primary submit buttons, brand icon (`.bi-house` in auth shell header), inline cross-link text, form input focus rings. Accent is NEVER used for non-interactive text, card borders, or decorative elements.

**Note for merged pages (dashboard.html extends main `base.html`):** The dashboard page inherits the main layout color system: dominant `#f8f9fa` (sidebar), secondary `#ffffff` (content area), accent `#0d6efd` (active nav).
