## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Wrong CDN integrity hash | auth/base.html used different (wrong) integrity hashes than main base.html. Browser silently refused CSS, breaking all auth page styling | blocking | Verify all CDN resource hashes match between templates or fetch correct hash via `openssl dgst -sha384 -binary \| openssl base64 -A` |
| Flask-SQLAlchemy 3.x relative DB path doubling | `sqlite:///instance/app.db` resolved to `<instance_path>/instance/app.db` because SA 3.x prepends instance_path. Use bare filename `sqlite:///app.db`. | advisory | Always use `sqlite:///filename.db` (no leading path) for SQLite dev DBs with Flask-SQLAlchemy 3.x |
| Server process killed on bash timeout | Background Flask processes die when bash tool invocation times out. Use `nohup` with redirects and always verify with a separate `curl` call afterward. | advisory | Start server with `nohup venv/bin/python run.py > /tmp/flask-nohup.log 2>&1 &` then `sleep 3 && curl localhost:5000` in separate bash call |
