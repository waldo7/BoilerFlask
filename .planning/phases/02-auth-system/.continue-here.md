## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Flask-SQLAlchemy 3.x relative DB path doubling | `sqlite:///instance/app.db` resolved to `<instance_path>/instance/app.db` because SA 3.x prepends instance_path. Use bare filename `sqlite:///app.db`. | advisory | Always use `sqlite:///filename.db` (no leading path) for SQLite dev DBs with Flask-SQLAlchemy 3.x |
